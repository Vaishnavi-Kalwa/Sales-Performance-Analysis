# Sales-Performance-Analysis
# Introduction

This project focuses on analyzing sales performance using a structured sales dataset and related dimension tables. The analysis was performed by building relationships between fact and dimension tables, calculating profitability, and visualizing insights through an interactive Excel dashboard to support business decision-making.

# Objective
The objective of this analysis is to provide business stakeholders with clear, actionable insights into sales performance and profitability. The dashboard is designed to help stakeholders:

Understand region-wise and country-wise sales contributions

Identify high-performing and underperforming markets

Monitor year-wise sales and quantity trends to support strategic planning

Evaluate profitability across sales channels (Wholesale, Distributor, Export)

Support data-driven decisions related to market focus, resource allocation, and revenue growth

# Description

The sales dataset includes the following key fields:

Order Date,

Sales Channel (Wholesale, Distributor, Export)

Warehouse Code, Delivery Region

Product Description Index

Order Quantity

Unit Price, Total Cost, Total Unit Cost

Additional related tables include:

Customer Index Table

Regions Table

Product Description Table

Relationships were created between these tables to form a proper data model. A Profit column was calculated using sales and cost metrics.
An interactive Excel dashboard was then developed to analyze:

Region-wise sales and quantity

Top 5 countries with the highest sales

Geographical sales distribution

Year-wise performance using a slicer

Total profit contribution

# Key Insights

Total Sales: $824 million.

Total Profit: $307.8 million.

Total Quantity Sold: 360,764 units.

Los Angeles recorded sales of $36.1 million, making it one of the top-performing regions.

In 2018, a significant decline in sales was observed:
Q3:$43.5 million sales with 18,933 units
Q4: $21.3 million sales with 9,180 units
(Data available only until November, contributing to the decline)

Wholesale channel generated higher profit compared to Distributor and Export channels, indicating stronger margins in wholesale operations.
